# ThePeriodicTable

この周期表は、Javascriptの授業で作成したものを、PHP学習中にクイズが出せるよう改良したものです。  
色分け、詳細表示、クイズ機能など、**7ファイルで構成されたシンプルな設計**です。

## 📁 ファイル構成
- `index.php`：メイン画面
- `header.php`：ヘッダー表示
- `footer.php`：フッター表示
- `quiz.php`：クイズ機能
- `js/elemnt.json`：元素データ
- `js/script.js`：動作スクリプト
- `css/style.css`：スタイル設定
